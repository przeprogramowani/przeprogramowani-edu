# P1 Brief: video-lesson-intro — Cel lekcji: kontrakt zamiast kodu

| | |
|---|---|
| Lekcja | m1-l1 |
| Sesja | 1 (rozgrzewka) |
| Format | Talking head + slajd z diagramem, PELNY prompter |
| Czas | 2-3 min |
| Ryzyko | 1/5 |
| Kolejnosc nagrania | **2 z 5** |

## Setup (kontynuacja z p1-welcome +)

- [ ] Slajd z diagramem lancucha gotowy:
  `Metna idea -> /10x-shape -> shape-notes.md -> /10x-prd -> prd.md -> Kolejne skille`
- [ ] PNG: `assets/diagrams/lessons-m1-l1-videos-video-lesson-intro-1-10x.png`
- [ ] Sprawdzic czytelnosc diagramu na pelnym ekranie
- [ ] Ton: pewny, rzeczowy, lekko prowokujacy ("Wiem, co myslisz..."), bez hype

## Tekst do promptera

> Skopiuj ponizszy blok 1:1. Tekst w `[nawiasach]` to wskazowki — nie czytaj na glos.

```
Masz pomysł na aplikację.
Brzmi sensownie — habit tracker, system notatek, panel dla freelancerów.
I odruchowo otwierasz edytor.

Po dwóch tygodniach okazuje się,
że nadal nie wiadomo, dla kogo to jest.
Jaki jest pierwszy przepływ. Co wypada z MVP.

Nie dlatego, że nie myślisz —
dlatego, że nikt ci nie zadał niewygodnych pytań.

W preworku mówiliśmy, że prompt to kontrakt
i poznałeś kryteria dobrego projektu.
Dzisiaj te dwie rzeczy łączą się w jedno:
Agent, działając w harnessie, weźmie twoje kryteria
i zacznie je egzekwować pytaniami.

[pauza — pokaż slajd z diagramem]

Cały łańcuch. Dwa skille, dwa artefakty.

/10x-shape prowadzi sesję.
Pyta, drąży, łapie luki.
Wynik — shape-notes.md. Zapis twoich decyzji.

/10x-prd bierze te notatki i zamienia w PRD o stałej strukturze.
Wiernie, bez domyślania się.
Czego brakuje — trafi do Open Questions.

To NIE jest generowanie dokumentu z mglistego promptu.
To zapisanie decyzji, które właśnie podjąłeś w rozmowie.
I od tej chwili kolejne skille nie czytają twojej intencji z powietrza.
Czytają artefakt na dysku.

[pauza — schowaj slajd, powrót do kamery]

Wiem, co myślisz.
Kolejny dokument przed kodowaniem.

Nie chodzi o dokument.
Chodzi o to, że po tej sesji będziesz miał decyzje —
konkretne, spisane i nie do podważenia kolejnym promptem.

Za chwilę zobaczysz to na żywo.
Uruchomię /10x-shape na projekcie 10xCards —
aplikacji, która generuje fiszki z twoich materiałów do nauki.
I pokażę, jak Agent wymusza precyzję — i co wychodzi z sesji.

Pierwszy ruch w tym kursie to rozmowa.
Zaczynamy.
```

## Po nagraniu

Sesja 1 zakonczona. Zmiana setupu na terminal (~10-15 min).
