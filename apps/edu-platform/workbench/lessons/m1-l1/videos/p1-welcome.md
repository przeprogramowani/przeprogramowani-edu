# P1 Brief: video-welcome — Powitanie + prework

| | |
|---|---|
| Lekcja | m1-l1 |
| Sesja | 1 (rozgrzewka) |
| Format | Talking head, PELNY prompter |
| Czas | 3-4 min |
| Ryzyko | 1/5 |
| Kolejnosc nagrania | **1 z 5** |

## Setup

- [ ] Kamera, oswietlenie, mikrofon
- [ ] Prompter zaladowany (tekst ponizej)
- [ ] Opcjonalnie: screenshot platformy z lista modulow preworku (overlay)
- [ ] Sprawdzic: czy quiz nadal dziala na platformie
- [ ] Ton: energiczny start, rzeczowy i cieply, bez hype

## Tekst do promptera

> Skopiuj ponizszy blok 1:1 do aplikacji promptera. Tekst w `[nawiasach]` to wskazowki — nie czytaj na glos.

```
Cześć, witajcie w pierwszej lekcji 10xDevs 3.0.

Prework za nami.
Zaczynamy budować.
Ale budowanie zaczyna się od decyzji, nie od kodu.

Zanim napiszesz pierwszą linię,
przeprowadzisz sesję z Agentem —
i ta sesja wymusi pytania, na które sam byś nie wpadł.

[pauza]

Ta lekcja stoi na preworku — na wszystkich czterech sekcjach.
Jeśli jeszcze nie wypełniłeś quizu na platformie — zrób to.
Pomaga nam dopasować agendę do twoich potrzeb.
Jeśli quiz chwilowo nie jest pod ręką, nie zatrzymuj się na tym teraz.
Ale do preworku wróć — serio.

[pauza]

W tym module zakładamy, że masz fundamenty.
Wiesz, czym różni się chatbot od Agenta i harnessu.
Masz skonfigurowane narzędzie AI.
Znasz też kryteria projektu zaliczeniowego.
Teraz złożymy to w pierwszy realny workflow.

[pauza]

Pierwszym ruchem w tym kursie nie będzie implementacja.
Będzie rozmowa — i to taka, po której zostaje coś na dysku.

Zaczynamy.
```
