### Alternative format under consideration: HTML + Tailwind page

Instead of (or alongside) generated illustrations, the cheatsheet could be a static HTML page styled with Tailwind. Rationale: far easier to iterate on content and style than re-prompting generated images — edit text/layout directly, searchable, responsive, embeddable, and version-controllable as a diffable artifact.
