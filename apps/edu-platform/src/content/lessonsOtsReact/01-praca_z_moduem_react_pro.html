Witaj w drugim module szkolenia Opanuj TypeScript: Frontend Pro. W tej lekcji dowiesz się jak efektywnie pracować z modułem React Pro, dzięki któremu zostaniesz ekspertem integrowania TypeScriptu z najpopularniejszą technologią frontendową na rynku jaką jest React.

Czeka na Ciebie wiele zwięzłych i konkretnych lekcji, wzbogaconych o ćwiczenia oraz quizy. Zanim przejdziesz do nauki, mamy dla Ciebie kilka informacji dotyczących pracy z repozytorium: opanuj-typescript-praktyka.

💡 Aktualizacja repozytorium

Przed startem pracy z drugim modułem upewnij się, że stan twojego forka repozytorium opanuj-typescript-praktyka jest aktualny.

Aby pobrać najnowsze zmiany, wejdź na stronę forka na GitHubie i klinij przycisk “Sync Fork” - dzięki temu pobierzesz najnowsze zmiany.

Po synchronizacji forka pobierz zmiany na twoją maszynę:

git pull

🚀 Ćwiczenia

W drugim module udostępniamy dla Ciebie zupełnie nowy zestaw ćwiczeń. Uruchomisz je w specjalnie przygotowanym sandboxie z Reactem i bibliotekami pomocniczymi. Aby dodać nowe zależności, wykonaj komendę:

npm install

Następnie możesz uruchomić serwer developerski polecenim:

npm run dev

Powinieneś teraz zobaczyć nową aplikację i linki do poszczególnych zadań:

👉 Ważne: W zależności od zadania, początkowy stan projektów może się różnić. Niektóre z nich wymagają nowych komponentów, a inne debugowania i naprawy istniejących.

Do zadań możesz też przechodzić bezpośrednio poprzez komendę:

npm run dev <task_name> # np. npm run dev survey

Tak jak wcześniej, do uruchomienia dedykowanych testów dla zadania wykorzystasz komendę “verify”:

npm run verify <task_name> # np. npm run verify survey

📝 Progress Tracker

Podobnie jak wcześniej, postępy możesz śledzić na dedykowanym dashboardzie:

npm run summary

Po uruchomieniu dashboardu powinieneś teraz zobaczyć nową sekcję dla modułu “O:TS React Pro” - trzymamy kciuki za komplet zielonych zadań!
